### 模块:Sirius    线程同步框架

- 静态模式
使用Sync注解标识同步方法，并且静态调用Sirius启动类,该方法会让同步方法一直在后台运行直到同步状态解除

- 动态模式
使用Sync注解调用标识同步方法，并在调用该方法时自动将方法使用线程运行
ASM字节码技术替换原类文件，

无损替换方式
1、根启动类，先生成同步类字节码文件，再使用自定义类加载器加载

2、 加载同步类字节码文件，不加载原类字节码
